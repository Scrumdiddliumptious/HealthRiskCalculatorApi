## Requirements

For development, you will only need Node.js installed in your environement.

## Setup

    $ git clone https://github.com/Scrumdiddliumptious/HealthRiskCalculatorApi.git
    $ cd HealthRiskCalculatorApi
    $ npm i

## Running the project

    npm run start
